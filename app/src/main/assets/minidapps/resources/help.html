MINIDAPPS
                  Powered by Minima

Creating a MiniDAPP is very easy.

They are simply static web sites, using standard HTML, JavaScript and CSS style sheets, coupled with the minima.js lib for interacting with the Minima network.

All the files are placed in a folder, zip it, and change the extension to '.minidapp'.

You can open and read any MiniDAPP you already have to see how it works. Just change the extension to .zip and you can view them like any other archive.

You can freely distribute MiniDAPPs, without the need to host them on costly centralised websites.

Without the worry of wondering if the website will be taken down.. if the developers will change your favourite DAPP.. if certain countries or areas will be censored.. or even if the service will be turned off altogether.

The minima.js file and complete sorce code are available from mifi.minima.global.. and locally at http://127.0.0.1:21000/minima.js

You can integrate Minima into your current web site or work-flow. This is known as MiFi.

You may want to accept payments, track and trace goods, distribute custom tokens that represent tickets or vouchers or even use Minima for managing in-game tokens.. anything is possible.

A MiniDAPP is slightly but powerfully different.

A MiniDAPP is when you integrate directly into Minima. The application becomes stand-alone and can be run by anyone, anywhere. They allow for truly decentralised applications that do not rely on ANY central party, service or authority. No website required.. :)

MiniDAPPs run directly from within the Minima application.

A completely uncensorable and unstoppable Ecosystem for Decentralised Applications.
